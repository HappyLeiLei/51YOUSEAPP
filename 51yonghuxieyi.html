51有色用户注册协议

1. 总则
1.1 为规范本平台交易活动，根据《中华人民共和国民法通则》、《中华人民共和国合同法》等法律、法规、政策及商务主管部门的有关规定，参照中华人民共和国国家标准 《大宗货物电子交易规范》（GB/T18769-2003），制定本协议。
1.2 本平台根据公平原则及诚实信用原则，依托本平台网站（http://www.51youse.com），运用安全、高效、便捷、先进的电子商务信息技术，为本平台会员提供铜铝相关产品的交易、交收及相关信息服务。
1.3 本协议适用于本平台内会员的注册及管理。
1.4 本平台有权依据本协议调整会员注册及管理的具体操作流程及要求。
2. 术语与定义
2.1 会员：是指具有合法经营资质，承认并遵守本平台各项管理规则，已在本平台注册、具有完全民事行为能力的企业法人及其他组织。
2.2 会员账号是指经会员注册申请，由本平台为会员开设的用于录入会员资料、查看平台信息、输入交易指令并在本平台进行货物交易活动的注册号码。
2.3 不可抗力是指不能预见、不能克服并不能避免且对本平台及会员造成重大影响的客观事件，包括但不限于自然灾害（如洪水、地震、瘟疫流行及风暴等）以及社会事件（如战争、动乱、政府行为等）。
3. 会员的权利与义务
3.1 会员的权利：
3.1.1 获得本平台提供的仅限于本平台使用的会员账号及密码；
3.1.2 有权指定代理人；
3.1.3 享受本平台提供的各项服务；
3.1.4 本平台规定会员可享有的其他权利。
3.2 会员的义务：
3.2.1 遵守国家法律、法规、政策、商业道德、各类合同及本平台各项规则等的规定，接受本平台的监督与管理，配合本平台的工作；
3.2.2 严格按本平台各项规则履行网站使用过程中的各项义务，并对自身的所有行为承担法律后果。
3.2.3 就会员向本平台提供的证件、签名、印鉴、文件等各种资料的真实性、合法性、有效性承担相应法律后果；如有变更，应及时联系本平台客服进行修改，并承担因未及时通知本平台作相应变更而导致的法律后果。
3.2.4 妥善保管账号及密码，并自行承担保管责任；
3.2.5 维护本平台声誉，协助本平台处理各种突发或异常事件；
3.2.6 承担发布虚假或带有误导性质的信息引起的一切后果；
3.2.7 其他应向本平台尽的义务及责任。
3.3 会员知悉并认可：本平台可以通过邮件、短信及电话等形式，向会员发送货物信息、促销活动等告知信息。
4. 会员管理
4.1 成为本平台会员应具备以下条件：
4.1.1 经工商或行政部门注册登记设立，且存续状态正常；
4.1.2 具备从事与铜铝及相关商品有关的生产、经营资质或消费条件；
4.1.3 具有固定的经营场所；
4.1.4 具有良好的资信及商誉；
4.1.5 本平台要求具备的其他条件
4.2 会员可通过会员账号查询本平台公示的各类信息，如需进行货物交易，须申请平台认证。由本平台进行资质及材料的审核，审核通过后，该会员正式获得本平台的交易资格。
4.3 会员如发生下列事项，应于发生之日起5个工作日内向本平台书面报告：
4.3.1 代理人变更；
4.3.2 法定代表人变更；
4.3.3 注册资本或股权结构变更；
4.3.4 名称、经营场所、经营范围及联系方式变更；
4.3.5 经营状况发生重大变化；
4.3.6 涉及重大诉讼案件、仲裁案件、行政处罚或查封、冻结等司法措施。
4.4 会员存在下列情况之一的，本平台有权暂停或取消其会员资格：
4.4.1 注销或被吊销营业执照；
4.4.2 工商信息登记为歇业、停业或异常；
4.4.3 企业经营存在重大风险，或被列入全国失信被执行人名单；
4.4.4 企业法定代表人或主要经营人员涉及刑事犯罪；
4.4.5 其他违反国家法律、法规、政策的事项；
4.4.6 其他依本平台管理规则认定的事项。
4.5 会员资格暂停或取消后，如已排除影响其会员资格的事由，可重新向本平台申请恢复会员资格。
4.6 会员合并、分立后新设立的法人若需承继会员资格，应向本平台提出申请。经本平台审查批准后，方可承继。
4.7 本平台向会员收取的服务费用标准详见各类书面合同及平台相关公告
5. 免责声明
5.1 本平台在现有技术水平及条件下维护平台服务的稳定性及安全性；但对于本平台不能及时预见、及时发现及防范的法律、技术以及其他风险，包括但不限于因自然灾害、社会事件等不可抗力及黑客攻击、计算机病毒、通信线路故障等因素导致的服务中断、数据丢失以及其他的损失及风险，本平台不承担由此引起的法律责任及经济损失。
5.2 本平台可以在发生本协议5.1所涉情形时采取对某种商品暂停交易或暂停本平台交易的决定，恢复交易时间以本平台公告为准，恢复交易时以暂停交易前最终记录的数据作为有效交易数据。对因上述原因本平台采取暂停交易引起的损失，本平台不承担任何赔偿责任。
5.3 若会员认为本平台涉及的作品、表演、录音录像制品，侵犯了会员的信息网络传播权或者删除、改变了会员的权利管理电子信息的，会员有权书面通知本平台，要求本平台删除该作品、表演、录音录像制品，或者断开与该作品、表演、录音录像制品的链接。
书面通知应当包含下列内容：
◆会员的身份信息（包括但不限于会员姓名、身份证、联系方式及地址等）；
◆要求删除或者断开链接的作品、表演、录音录像制品的名称及网络地址；
◆要求删除或断开链接的理由及证明作品、表演、录音录像制品存在侵权的相关材料；
◆关于内容真实性的声明：“本人保证，本通知中所述信息是充分、真实、准确的，若本通知内容不完全属实，本人将承担由此产生的一切法律后果”；
◆签署会员的有效签章。
本平台收到会员的书面通知后删除该作品、表演、录音录像制品，或者断开与该作品、表演、录音录像制品的链接的，视为本平台已经尽到了合理的义务，本平台不承担任何其他赔偿责任。
5.4 本平台重视对会员保密信息的维护，但因本协议5.1所涉及情况或其他非本平台原因引起的会员保密信息泄露，本平台不承担任何赔偿责任。
6. 信息发布及管理
6.1 本平台网站用于公布平台规则、公告、交易货物行情及其它统计资料及信息。
6.2 电话、传真、信函、电子邮件、手机短信及本平台网站的公告及通知均为本平台向会员传递信息的有效途径。对于已经发出的信息均视为向会员的有效送达。
6.3 本平台根据本协议做出的各种决定、规则调整等，均通过本平台网站对外公布，除另有规定外，自公布之日起生效。
7. 违约与违规处理
7.1 会员对其账号在本平台的一切行为承担全部法律后果。
7.2 对违约或违反本平台有关规定的会员，本平台有权视情节轻重对其进行警告、暂停或关闭会员账号，并在有关网站及媒体上进行公告。违规行为涉嫌犯罪的，本平台有权采取移交司法机关并追究刑事责任等措施。通过上述措施不足以弥补本平台损失的，本平台有权通过法律途径进一步追究会员的责任。
7.3 会员对于交易过程中接触到的本平台商业秘密承担保密义务，会员不得泄露或非法使用本平台商业秘密。会员泄漏或非法使用商业秘密的，应承担由此引起的一切法律后果；涉嫌犯罪的，本平台有权采取移交司法机关并追究刑事责任等措施。
